# NetKeeperServer
安卓手机端自动发送短信获取闪讯密码并回传给请求方

## 前提说明

该服务端与[NetKeeperHelper](https://github.com/BeckXuan/NetKeeperHelper)配套使用

该服务端需要手动编译，不提供apk文件

使用kotlin编写

建议将手机设置为静态ip

## apk说明

apk中未提供详细的界面

参数均只在源码中修改

安装apk后请授予完整的权限

## 相关参数配置

请在编译apk前手动配置NetkeeperSever.kt中有注释说明的参数
